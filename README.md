# 坦克大战 🎮 Battle City

一个使用纯 HTML/CSS/JavaScript 构建的经典 NES 坦克大战（Battle City）克隆版。

## 游戏特点

- 🎯 **3个独特关卡** - 每个关卡有不同的地图布局和难度
- 🤖 **智能敌人AI** - 敌人会追踪你的基地并射击
- 🏆 **分数系统** - 每消灭一个敌人获得 100 分
- ❤️ **生命系统** - 初始 3 条命
- 🛡️ **出生保护** - 重生时短暂无敌
- 💥 **特效** - 爆炸特效、屏幕震动
- 🔊 **音效** - 射击、爆炸、过关音效
- 📱 **触屏支持** - 移动设备上也可游玩

## 操作方式

| 按键 | 操作 |
|------|------|
| **W / ↑** | 向上移动 |
| **S / ↓** | 向下移动 |
| **A / ←** | 向左移动 |
| **D / →** | 向右移动 |
| **空格** | 射击 |
| **Enter** | 开始游戏 |

## 游戏规则

- 保护基地（右下方的旗帜）不被敌人摧毁
- 消灭所有敌人即可过关
- 共有 3 个关卡
- 敌人有不同类型：普通、快速、装甲

## 如何运行

直接打开 `index.html` 文件即可在浏览器中运行，无需任何外部依赖。

## 技术栈

- HTML5 Canvas
- 原生 JavaScript
- CSS3
- Web Audio API（音效生成）
